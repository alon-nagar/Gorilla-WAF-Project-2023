About Gorilla Admin Manager

Gorilla Admin Manager is a web application that allows you to manage the Gorilla WAF.
You can view and change the incoming requests, blacklisted IPs, view statistics and change the settings.


Features

-> View the incoming requests.
-> Add/Remove blacklisted IPs
-> View statistics.
-> Start/Stop the WAF.
-> Add/Remove allowed URLs to redirect.

About Gorilla WAF

Gorilla is a powerfull Web-Application Firewall (WAF), that detect and block different web attacks.
It communicates with a local MongoDB database that saves the attacker's IP address, and sits behind a Nginx server in Linux environment.


Supported Attacks

-> Cross-Site Scripting (XSS) & HTML Injection (HTMLi).
-> SQL Injection (SQLi) [Note: Detection occurs with Machine Learning].
-> HTTP Parameter Pollution (HPP).
-> SSI Injection (SSIi).
-> Open Redirect Vulnerability.
-> HTTP Host Header Injection (HHHi).
-> Cross-Site Tracing (XST).
